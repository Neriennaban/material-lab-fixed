
1. Проверить параметры в пресете:
   - Composition: Fe-0.45%C

✅ **Критерии acceptance:**
- [ ] Изображение 2048×2048 пикселей
- [ ] Видны границы зёрен

❌ **Если не прошло:**
- Увеличить contrast в synthesis_profile
- Уменьшить artifact_level
- Перегенерировать с другим seed

## 🧮 ШАГ 5: Расчёт ASTM номера

### 5.1 Расчёт эквивалентного числа зёрен

**Формула:**
```
N = P + Q/2
```

**Пример:**
```
P = 32, Q = 24
N = 32 + 24/2 = 32 + 12 = 44
```

### 5.2 Расчёт плотности зёрен

**Формула:**
```
N_A = N / S
```

где S — площадь сетки в мм²

**Пример:**
```
N_A = 44 / 5000 = 0.0088 зёрен/мм²
```

### 5.3 Пересчёт на увеличение 100x

**Формула:**
```
n = N_A × 645.16
```

(645.16 мм² = 1 дюйм²)

**Пример:**
```
n = 0.0088 × 645.16 = 5.68 зёрен/дюйм²
```

### 5.4 Расчёт номера ASTM

**Формула:**
```
G = 1 + log₂(n)
```

**Пример:**
```
G = 1 + log₂(5.68) = 1 + 2.51 = 3.51
```

**Итоговый результат:**
```
ASTM G = 3.51 ± 0.5
```

## 📊 ШАГ 6: Расчёт предела текучести

### 6.1 Формула Холла-Петча

```
σ_y = σ_0 + k_y × d^(-1/2)
```

### 6.2 Исходные данные

| Параметр | Значение |
|----------|----------|
| σ_0 (для феррита) | 50 МПа |
| k_y (для феррита) | 0.50 МПа·м^0.5 |
| d (размер зерна) | ___ мкм (из измерений) |

### 6.3 Конвертация единиц

```
d (м) = d (мкм) × 10^(-6)
```

**Пример для d = 65 мкм:**
```
d = 65 × 10^(-6) = 6.5 × 10^(-5) м
```

### 6.4 Расчёт

**Шаг 1:** Вычислить d^(-1/2)
```
d^(-1/2) = 1 / √d = 1 / √(6.5 × 10^(-5)) = 1 / 0.00806 = 124.0 м^(-0.5)
```

**Шаг 2:** Подставить в формулу
```
σ_y = 50 + 0.50 × 124.0 = 50 + 62.0 = 112.0 МПа
```

**Итоговый результат:**
```
Предел текучести σ_y = 112.0 МПа
```

### 6.5 Автоматический расчёт (Python)

```python
from core.mechanical_properties_calculator import calculate_yield_strength_hall_petch

# Ввести размер зерна в мкм
d_um = 65  # ваше значение

# Расчёт
result = calculate_yield_strength_hall_petch(grain_size_um=d_um, material="ferrite")

# Вывод результатов
print(f"Размер зерна: {d_um} мкм")
print(f"Предел текучести: {result['yield_strength_mpa']} МПа")
print(f"Формула: {result['formula']}")
```

## 🆘 УСТРАНЕНИЕ НЕИСПОВНОСТЕЙ

### Проблема: Не видны границы зёрен

**Причины:**
- Слабое травление
- Низкий контраст

**Решение:**
1. Увеличить время травления (10→15 секунд)
2. Увеличить contrast в synthesis_profile (1.12→1.2)
3. Перегенерировать образец

### Проблема: Невозможно сфокусировать

**Причины:**
- Исчерпан диапазон фокусировки

**Решение:**
1. Сбросить фокусировку (кнопка [Reset])
2. Начать с грубой фокусировки
3. Проверить, выбран ли правильный объектив (10x)

### Проблема: ASTM не сходится с ожидаемым

**Причины:**
- Мало зёрен для статистики
- Неправильный подсчёт P и Q

1. Увеличить область измерений
2. Пересчитать зёрна (проверить с одногруппником)
3. Использовать метод сравнения для проверки
